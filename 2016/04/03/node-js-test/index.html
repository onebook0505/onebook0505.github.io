Javascript 常用測試 lib

  • Mocha ( 具靈活性的測試框架 : describe、beforeEach、afterEach、it、done )
  • Chai ( TDD/BDD 斷言風格 )
  • Sinon ( spy、stub、mock )
  • Zombie ( View 行為測試 )
  • Supertest ( Http 測試 )
  • faker ( 模擬數據 )

持續集成

持續集成 Continuous integration = 程式碼更新就整個 Run 看看是否出錯．
推薦使用 : Travis ( 開源專案免費 )、CircleCI．

  • 程式人員可以更腳踏實地的寫程式
  • PM 可以更好的掌握時程確保穩定性
  • 組織可以降低成本與開發風險
  • 客戶可以拿到品質優良的產品
