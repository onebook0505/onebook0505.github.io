node.js 開發環境建置

發表於 2016-03-20   |  


適用於 MAC OS 系統

  1. 安裝 homebrew ( mac os 套件安裝工具 ) ．
  2. 在 terminal 下 brew install nvm 安裝 nvm 用於管理與安裝 nodejs 各版本．( 記得加入bash_profile )．
  3. 在 terminal 下 brew install mongodb 安裝 mongodb．
  4. 利用 robomongo 提供的介面管理 mongodb．

參考資料 : 安裝NVM

其他

  • nodemon 監控程式碼更新與重啟 server ( 開發用 )
  • nginx 部署 ( 上線用 )
  • pm2 保持程序永不中斷運行 ( 上線用 )
  • mongolab ( 上線用，mongodb 雲端服務 )
